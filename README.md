# pages-siteindex-leyu

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要存储一系列独立的 HTML 页面，用于展示各类静态内容。所有页面均为手动整理的归档文件，不绑定特定域名或网站，适合作为页面资源集合使用。

## 目录说明

```
pages-siteindex-leyu/
├── index.html          # 主索引页面
├── pages/              # 归档页面存放目录
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
└── assets/             # 公共资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：仓库的入口页面，提供所有归档页面的导航链接。
- `pages/`：存放所有独立 HTML 页面，文件名通常反映页面内容或用途。
- `assets/`：存放页面间共享的样式、脚本和媒体文件，便于统一维护。

## 页面归档说明

- 所有页面均为纯静态 HTML，不依赖后端服务。
- 页面间相互独立，可单独访问，也可通过 `index.html` 统一浏览。
- 归档内容可能随时间更新或补充，历史版本保留在 Git 提交记录中。

## 维护说明

- 如需添加新页面，请在 `pages/` 目录下创建 HTML 文件，并在 `index.html` 中添加对应链接。
- 公共资源修改后，请确保不影响已有页面的正常展示。
- 建议每次提交附带清晰的提交信息，说明新增或修改的内容。

## 许可

本仓库内容仅供归档与展示，不涉及特定用途声明。如有引用外部资源，请遵守相应授权协议。
